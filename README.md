## 🚀 Getting Started

### Run Locally
1. Clone the repo:
   ```bash
   git clone https://himanshu07ghosh.github.io/Elementum-Frontend-Website/
   ```
2. Open the folder:
   ```bash
   cd elementum
   ```
3. Open `index.html` in your browser — done!

Or in VS Code terminal:
```bash
code .
```

---

## ☁️ Deploy on Vercel

1. Push this project to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Leave all settings as default (no build command needed)
5. Click **Deploy** ✅
